# qBittorrent 自动上传工具

这是一个GUI工具，可以自动将指定目录中的所有 `.torrent` 文件上传到 qBittorrent，并可以指定分类。

## 功能特点

- ✅ 图形化界面，操作简单
- ✅ 自动扫描目录下所有 `.torrent` 文件（包括子目录）
- ✅ 批量上传到 qBittorrent
- ✅ 支持自定义分类
- ✅ 支持 qBittorrent Web UI 认证
- ✅ 实时日志显示

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用前准备

1. 确保 qBittorrent 正在运行
2. 启用 qBittorrent 的 Web UI：
   - 打开 qBittorrent
   - 进入 `工具` → `选项` → `Web UI`
   - 启用 `Web 用户界面`
   - 记住端口号（默认 8080）
   - 如果需要，设置用户名和密码

## 使用方法

1. 运行程序：
   ```bash
   python main.py
   ```

2. 配置 qBittorrent 连接：
   - URL: 默认为 `http://localhost:8080`（根据你的设置修改）
   - 用户名和密码（如果设置了的话）
   - 点击"连接"按钮

3. 选择种子目录：
   - 点击"浏览"按钮选择包含 `.torrent` 文件的目录

4. （可选）设置分类：
   - 在"分类"输入框中输入分类名称

5. 扫描种子文件：
   - 点击"扫描种子文件"按钮，程序会扫描目录中所有的 `.torrent` 文件

6. 开始上传：
   - 点击"开始上传"按钮，程序会批量上传所有种子文件

## 注意事项

- 程序会递归扫描选中目录及其所有子目录中的 `.torrent` 文件
- 上传过程中可以看到实时日志
- 如果某个文件上传失败，会在日志中显示错误信息，但不影响其他文件的上传
